Detecção de Tipos de Roupas com Redes Neurais

Este é um projeto pessoal em que desenvolvi uma rede neural para classificar imagens de roupas em diferentes categorias, usando o conjunto de dados Fashion MNIST como base.

Descrição
Treinei o modelo para reconhecer 10 categorias de roupas, como camisetas, calças e sapatos. Além de testar o modelo com as imagens do Fashion MNIST, experimentei usar uma imagem externa de uma sandália para verificar como ele se comporta com dados reais.

Funcionalidades
Treinamento e validação do modelo com os dados do Fashion MNIST.

Avaliação do desempenho em termos de precisão e perda.

Teste prático com imagens externas, como a sandália.

Geração de gráficos para visualizar o histórico de treinamento.
